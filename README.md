# containers
ft_containers

Учебный проект по изучению STL.

Воспроизведено несколько библиотечных контейнеров на С++98 в соответствии с заданием [en.subject.pdf](./en.subject.pdf):
* vector
кроме `vector<bool>` .
* map (красно-черное дерево)
* stack

А также:
* iterators_traits
* reverse_iterator
* enable_if
* is_integral
* lexicographical_compare
* std::pair
* std::make_pair
